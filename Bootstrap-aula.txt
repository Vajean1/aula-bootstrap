Bootstrap

O que é:

- Se trata de um framework para desenvolvimento de sites responsivos
-Possui uma vasta biblioteca de componentes que facilita e agiliza o desenvolvimento de páginas web
-É um dos frameworks mais conhecidos e usados para desenvolvimento front-end

----------------
O bootstrap separa a tela em 12 colunas, seguindo as classes:
 conteiner - conteiner

acessando a biblioteca do bootstrap encontra-se
todos as classes que da pra por nas tags e formata-las

container-fluit = deixa o conteiner completo na pagina
text-center = alinha o texto no centro
